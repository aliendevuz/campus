# Contribution

Loyiha ishtirokchilari loyihaga hissa qo'shish madaniyatlariga amal qilishi loyihani sifatini buzib qo'ymaslik uchun muhim

ayniqsa git bilan ishlash jarayonida har doim tekshirilishi kerak bo'lgan narsalar:
- biriktirilgan task bilan tanishib chiqish
- taskni `in review` ga olib o'tish
- `git pull origin main` kommandasi orqali loyihani eng so'nggi barqaror versiyasiga hissa qo'shish

yana vibe coding bo'yicha ham ko'rsatmalar yozib chiqishim kerak!
